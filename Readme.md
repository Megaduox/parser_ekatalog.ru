## Парсер сайта e-katalog.ru

Скрипт парсинга листинга товаров с сайта e-katalog.ru

## Запуск

```python
python main.py
```

Глобальная переменная URL - ссылка на категорию каталога, которую будет
парсить скрипт. Формат ссылки такой https://www.e-katalog.ru/list/431/.

На выходе скрипт формирует csv-файл с результатами парсинга со списокм товаров
со следующими полями: Name, Title, Url, Price.
Файл располагается в корневой дирректории скрипта.


## Прочее

Скрипт написан в учебных целях.

Со временем вёрстка сайта может поменяться и нужно будет доработать xpath-выражения.
